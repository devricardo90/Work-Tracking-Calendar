# App Sitemap

## Principal Structure
- **Login / Access (optional)**
    - Sign In
    - Create Account

- **Home / Calendar**
    - Monthly Calendar View
    - Previous Month
    - Next Month
    - Select a Day
    - Quick Add Entry

- **Day Entry / Day Details**
    - Date
    - Hours Worked
    - Location
    - Notes
    - Save Entry
    - Edit Entry
    - Delete Entry

- **Monthly Summary**
    - Worker Name
    - Selected Month
    - Total Monthly Hours
    - Total Worked Days
    - Average Hours per Day
    - Entries List
    - Export as PDF
    - Send by Email

- **History / Records**
    - Search by Date
    - Filter by Month
    - Filter by Location
    - View Past Entries

- **PDF Export**
    - Preview PDF
    - Download PDF
    - Share by Email

- **Profile / Settings**
    - Worker Name
    - Email
    - Default Work Hours (optional)
    - Saved Locations
    - Language

## Recommended Navigation
For mobile, the ideal navigation is a bottom navigation bar with 4 items:
1. **Calendar**
2. **History**
3. **Summary**
4. **Profile**

*Add Entry button can be a Floating Action Button (+) or open when tapping a day on the calendar.*

## Usage Hierarchy
Calendar → Select Day → Add/Edit Entry → Save → View Saved Day → Check Monthly Summary → Export PDF → Send by Email
